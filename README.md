
# RECONHECIMENTO AUTOMÁTICO DE SINAIS EM LIBRAS NA ÁREA DA TECNOLOGIA DA INFORMAÇÃO: ESTRUTURAÇÃO DE UMA BASE DE DADOS E AVALIAÇÃO DE UM MODELO LSTM.

Este repositório contém o conjunto de dados (vídeos e Landmarks extraídos) desenvolvido no âmbito do Trabalho de Conclusão de Curso , submetido à Universidade Federal Rural do Semi-Árido (UFERSA).

## Descrição do Projeto

O objetivo principal deste trabalho foi a estruturação de uma base de dados inédita voltada ao vocabulário técnico de Tecnologia da Informação (TI) na Língua Brasileira de Sinais (Libras), englobando 30 termos distintos obtidos no canal Libras Ifsc Chapecó. Utilizou-se o framework Google MediaPipe (módulos Hands e Holistic) para extrair as Landmarks.
